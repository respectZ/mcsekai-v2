# mcsekai
Play a certain 3dmv rhythm game in minecraft bedrock edition.

Song actors are fully configurable.

# Quick Start
- Download the Release
- Install the pack
- Add it into the world
- Find a flat place
- `summon mcsekai:song_manager` and interact it.
